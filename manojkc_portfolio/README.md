# Manoj K.C. | Backend Developer Portfolio

A responsive, modern, and dynamic personal developer portfolio built to showcase my backend projects, technical skills, and professional experience. 

![Portfolio Preview](assets/images/profile.jpg) 

## 🚀 Key Features

* **Modern UI:** Responsive design built with HTML5 and Tailwind CSS for a seamless experience across desktop and mobile devices.
* **Dynamic Backend:** Powered by Python and FastAPI to serve static files, assets, and handle backend routing.
* **Functional Contact Form:** Includes a working contact endpoint that securely processes user messages.
* **Database Integration:** Contact form submissions are saved to a local SQLite database utilizing SQLAlchemy ORM.
* **Automated Emails:** Integrated with `smtplib` to instantly send email notifications to my inbox whenever a new form is submitted.
* **Resume Access:** Direct download link for my latest professional CV.

## 🛠️ Technologies Used

**Frontend:**
* HTML5
* Tailwind CSS
* JavaScript (ES6+)

**Backend:**
* Python 3
* FastAPI
* Uvicorn (ASGI web server)

**Database & Data Validation:**
* SQLite
* SQLAlchemy (ORM)
* Pydantic
